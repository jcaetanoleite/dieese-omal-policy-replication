# Contributing

Please open an issue before changing the empirical definition of OMAL. A proposed methodological change should identify:

1. the expenditure concept affected;
2. the POF variable or translator code used;
3. the IPCA group, item or subitem used for updating;
4. the expected effect on city estimates;
5. whether the change breaks comparability with the published article.

Run `pytest -q` before submitting a pull request.
